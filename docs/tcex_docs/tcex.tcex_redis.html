tcex.tcex_redis module¶

TcEx Framework Redis Module

class tcex.tcex_redis.RedisClient(host='localhost', port=6379, db=0, blocking_pool=False, **kwargs)[source]¶

Bases: object

A shared REDIS client connection using a ConnectionPooling singleton.

Initialize a single shared redis.connection.ConnectionPool. For a full list of kwargs see https://redis-py.readthedocs.io/en/latest/#redis.Connection. :param host: The REDIS host. Defaults to localhost. :type host: str, optional :param port: The REDIS port. Defaults to 6379. :type port: int, optional :param db: The REDIS db. Defaults to 0. :type db: int, optional :param blocking_pool: Use BlockingConnectionPool instead of ConnectionPool. :type blocking_pool: bool :param errors: The REDIS errors policy (e.g. strict). :type errors: str, kwargs :param max_connections: The maximum number of connections to REDIS. :type max_connections: int, kwargs :param password: The REDIS password. :type password: str, kwargs :param socket_timeout: The REDIS socket timeout. :type socket_timeout: int, kwargs :param timeout: The REDIS Blocking Connection Pool timeout value. :type timeout: int, kwargs
